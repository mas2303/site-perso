Installing macOS Sonoma on Xiaomi Notebook Pro

27 August 2024 - By Friendshy

I

n this article, I'll guide you through the steps to install macOS on the Xiaomi Notebook Pro. Thanks to the Hackintosh community, it’s now possible to turn your Xiaomi laptop into a fully functional Mac. For this tutorial, I used the EFI available on daliansky's GitHub repository and the macOS Sonoma Internet Recovery Install method.

Prerequisites
Before getting started, make sure you have:
  • A Xiaomi Notebook Pro.
  • A USB drive with at least 16 GB capacity.
  • Another Mac or PC with macOS to prepare the bootable USB.
  • The daliansky's EFI file.
  • A stable internet connection for the recovery installation.
Steps to Install macOS Sonoma on Xiaomi Notebook Pro

Step 1: Preparing the Bootable USB Drive

  • 01 - Download macOS Sonoma:
    • - If you have access to a Mac, download macOS Sonoma from the App Store.
    • - If you’re using a PC, you can download the system image from Apple’s website or use other online sources.
    • - Or the method I recommend: Online method.
  • 02 - Create the Bootable USB Drive:
    • - Insert the USB drive into your Mac or PC.
    • - Format the USB drive using Disk Utility (Mac) or Diskpart (Windows), selecting Mac OS Extended (Journaled) as the format and GUID Partition Map as the scheme.
    • - Open Terminal and run the following command to create the bootable USB:
      'sudo /Applications/Install\ macOS\ Sonoma.app/Contents/Resources/createinstallmedia --volume /Volumes/USB --nointeraction'
    • - Replace '/Volumes/USB' with the name of your USB drive.
  • 03 - Download and Install the EFI:

Step 2: Download the EFI folder from daliansky's GitHub and copy it to the EFI partition of your USB drive.

  • 01 - Access the BIOS:
    • - Restart your computer and press 'F2' to enter the BIOS.
  • 02 - BIOS Settings:
    • - Disable Secure Boot.
    • - Enable UEFI Boot.
    • - Ensure SATA Mode is set to AHCI.
  • 03 - Boot Priority:
    • -Set the USB drive as the first boot device.

Step 3: Installing macOS

  • 01 - Boot from the USB Drive:
    • - Insert the USB drive into the Xiaomi Notebook Pro.
    • - Restart and press F12 to access the boot menu, then select the USB drive.
  • 02 - Using Internet Recovery:
    • - Once the macOS installation menu loads, select Disk Utility to format your main hard drive to APFS.
    • - Return to the main menu and select the Internet Recovery option.
    • - Follow the prompts to install macOS Sonoma.

Step 4:Post-Installation

  • 01 - Initial Setup:
    • - After installation, your Xiaomi Notebook Pro will restart automatically. Complete the initial setup of macOS just as you would with a native Mac.
  • 02 - Updating the EFI:
    • - It’s recommended to regularly check for EFI updates on daliansky’s GitHub repository and apply them to ensure the system runs smoothly.
  • 03 - Installing Additional Kexts:
    • -Install any additional kexts needed for your specific model of Xiaomi Notebook Pro. These kexts can be found in the downloaded folder or through Hackintosh forums.
Conclusion

By following these steps, you should be able to install and use macOS on your Xiaomi Notebook Pro. Remember, while this process is exciting, it requires patience and some technical know-how. With the right setup, you’ll have a powerful laptop capable of running macOS without issues. Enjoy your new Hackintosh!
